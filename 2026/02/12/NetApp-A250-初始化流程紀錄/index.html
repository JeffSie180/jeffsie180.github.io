NetApp A250 初始化流程紀錄

發表於2026-02-12|更新於2026-02-12|NetApp
|總字數:736|閱讀時間:4分鐘|瀏覽量:
NetAppONTAP

前言

本文記錄 NetApp A250 搭配 ONTAP 9.10 的初始化流程，涵蓋磁碟分割（Advanced Drive Partitioning）、Cluster Setup Wizard 以及時區與 NTP 設定。

環境與前置條件

| ONTAP Version: 9.10

1. LOADER-A 進入 ONTAP menu

選擇 (9) Configure Advanced Drive Partitioning。

2. 選擇 (9a) Unpartition all disks

選擇 (9a) Unpartition all disks and remove their ownership information。

ONTAP Version: 9.12 會多一個確認提示：
Do you want to abort this operation(yes/no)? no

3. LOADER-B 執行 (9a)

LOADER-A 做完 (9a) 後，接著在 LOADER-B 執行相同步驟。

4. LOADER-A 選擇 (9b) 初始化磁碟分割

選擇 (9b) Clean configuration and initialize node with partitioned disks。

5. LOADER-B 執行 (9b)

LOADER-A 做完 (9b) 後，接著在 LOADER-B 執行相同步驟。

6. 初始化完成 — Cluster Setup

兩個節點都完成後，系統會進入 Cluster Setup Wizard。

7. 設定時區（重要！）
